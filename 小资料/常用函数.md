# 常用函数

### 数字转字符串(int->string)

```c++
string Change1(int x){
    if(x==0) return "0";
    string A;
    while(x){
        A+=((x%10)+'0');
        x/=10;
    }
    reverse(A.begin(),A.end());
    return A;
}
```

注：其中 $x$ 为非负整数，返回的 $A$ 为字符串。

------

### 字符串转数字(string->int)

```c++
int Change2(string x){
    int cnt=0;
    for(int i=0;i<(int)x.size();i++){
        cnt*=10;
        cnt+=(x[i]-'0');
    }
    return cnt;
}
```

注：其中 $x$ 为非空字符串，返回的 $cnt$ 为非负整数。

### 判断质数

```c++
bool prime(int a)
{
    if(a<=1) return 0;
    bool f=1;
    for(int i=2;i*i<=a;i++) if(!(a%i)) f=0;
    return f;
}
```

注：其中 $a$ 为非负整数。函数返回值为 1(true)  时，则 $a$ 为质数，返回值为 0(false) 时，则 $a$ 不为质数。

### 判断某个字符是什么

```C++
int check(char x){
    if(x>='0'&&x<='9') return 1;//x是数字
    if(x>='A'&&x<='Z') return 2;//x是大写字母
    if(x>='a'&&x<='z') return 3;//x是小写字母
    return 0;//x既不是数字也不是字母
}
```

### 固定占位输出

```c++
cout<<setw(n)<<setfill(m)<<x;
```

注：setw(n) 表示该次输出固定占 $n$ 位，setfill(m) 表示若输出量不够 $n$ 位，不够的位用 $m$ 字符填充(此处 $m$ 为一个 char 类型变量)，$x$ 表示要输出的内容。

**例一：**

```C++
int n=5;char m='0';
cout<<setw(n)<<setfill(m)<<123;
/*
输出：
00123
*/
```

**例二：**

````c++
int n=4;char m=' ';
cout<<setw(n)<<setfill(m)<<"ab";
/*
输出：
  ab
*/
````

### 数组中的常用函数(以 int 数组为例)

```c++
int a[100005];
sort(a+1,a+1+n)//把a[1~n]内所有数从小到大排序
sort(a,a+n)//把a[0~(n-1)]内所有数从小到大排序
reverse(a+1,a+1+n)//翻转a[1~n]内所有数，翻转后的a[1]为原先a[n]的值，a[2]为原先a[n-1]的值 
```
